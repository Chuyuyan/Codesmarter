## üí° Pro Tips

1. **Be Specific**: More specific questions get better answers
   - ‚ùå "How does it work?"

2. **Ask Follow-ups**: The chat remembers context
   - Ask: "What is this project?"
   - Then ask: "Explain the main components"

3. **Use Analysis Types**: Change the dropdown to get different types of responses

   - Refactor: Improvement suggestions
   - Debug: Finding issues
   - Optimize: Performance tips
